Detailed Description

(PRIVATE) The firmware version from a bulb

This is the state firmware payload returned from the bulb when GET_HOST_FIRMWARE is requested.

The firmware is dotted decimal, where the major is a different number for each bulb type, and the minor is the actual firmware revision.

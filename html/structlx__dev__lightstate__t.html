Detailed Description

(PRIVATE) Color response from a bulb

This is the state container returned from the bulb when GET_COLOR is requested. This struct will be parsed and stored as the color structure, which is what is sent.

LIFX colors are HSBK, not HSV or RGB, so brightness here is V in HSV and Kelvin is generally left at 3500 for all operations, but can be changed.

\doxysubsection{Detailed Description}
An \mbox{\hyperlink{classHSBK}{H\+S\+BK}} container class with helper functions for direct use of the L\+I\+FX color scheme. 

Attempts to abstract the \mbox{\hyperlink{classHSBK}{H\+S\+BK}} concept, and allow for easy assignment and management of L\+I\+FX specific \mbox{\hyperlink{classHSBK}{H\+S\+BK}} colors. L\+I\+FX uses H\+SV, but replaces V with B for brightness.

\mbox{\hyperlink{classHSBK}{H\+S\+BK}} is Hue, Saturation, Brightness, and Kelvin. It maps directly to H\+SV, but includes the Kelvin argument. When assigning the \mbox{\hyperlink{classHSBK}{H\+S\+BK}} to a Q\+Color, Kelving is ignored unfortunately. Therefore, it\textquotesingle{}s not possible to map 1-\/1. I\textquotesingle{}m still looking into how to \char`\"{}fix\char`\"{} this.

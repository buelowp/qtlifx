\doxysubsection{Detailed Description}
This is the state container returned from the bulb when G\+E\+T\+\_\+\+C\+O\+L\+OR is requested. This struct will be parsed and stored as the color structure, which is what is sent.

L\+I\+FX colors are H\+S\+BK, not H\+SV or R\+GB, so brightness here is V in H\+SV and Kelvin is generally left at 3500 for all operations, but can be changed.

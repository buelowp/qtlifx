\doxysubsection{Detailed Description}
(P\+U\+B\+L\+IC) Lifx Group container 

This is a container for a L\+I\+FX group. It is able to store and retrieve any bulb in the group and provides group details. Access to the individual bulbs then is possible by asking for a bulb pointer and referencing it\textquotesingle{}s content. This is the same pointer used by the manager class and can be used the same way.
